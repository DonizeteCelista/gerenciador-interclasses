GERENCIADOR DE INTERCLASSES 1.0 — ATUALIZAÇÃO WEB

ALTERAÇÕES
- Eliminatórias compactas para qualquer quantidade de participantes.
- 10 participantes: somente 4 jogam uma preliminar; 6 avançam direto às quartas.
- 19 participantes: somente 6 jogam os 16 avos; 13 avançam direto às oitavas.
- Acima de 32: cria 32 avos quando necessário.
- As fases aparecem da esquerda para a direita, antes das oitavas, quartas, semifinal e final.
- Uma fase só é liberada depois que todos os jogos da anterior forem encerrados.
- O número do jogo é fixo e não muda quando resultados são adicionados.
- Painel de placar horizontal: TIME A 0 × 0 TIME B.
- Botão para inverter os lados da quadra/campo.
- Esta atualização é somente da versão WEB. O APK será retomado depois dos testes.

COMO ATUALIZAR O GITHUB
1. Abra o repositório gerenciador-interclasses.
2. Substitua:
   index.html
   manifest.json
   sw.js
   icon.svg
   .nojekyll
3. Faça o commit.
4. Aguarde o Pages ficar verde.
5. Abra o site com Ctrl + F5 ou em janela anônima.
6. Para testar a nova eliminatória, crie uma competição nova.
