GERENCIADOR DE INTERCLASSES 1.0 — PACOTE DEFINITIVO PARA GITHUB PAGES

Este pacote foi simplificado para evitar o problema do site sem aparência.

ARQUIVOS:
- index.html: contém HTML, CSS e JavaScript.
- manifest.json
- sw.js
- icon.svg
- .nojekyll

COMO SUBSTITUIR NO GITHUB

1. Abra o repositório gerenciador-interclasses.
2. Apague os arquivos antigos do site:
   index.html
   style.css
   app.js
   firebase.js
   dados.js
   manifest.json
   sw.js
   icon.svg
3. Não precisa apagar README.md.
4. Clique em Add file > Upload files.
5. Envie:
   .nojekyll
   index.html
   manifest.json
   sw.js
   icon.svg
6. Faça o Commit na branch main.
7. Espere o pages build and deployment ficar verde.
8. Abra:
   https://donizetecelista.github.io/gerenciador-interclasses/
9. Pressione Ctrl + F5.
10. Se aparecer a versão antiga, abra em janela anônima.

O index.html não depende de pastas css, js ou assets.
